# گالری شخصی نقاشی

آرشیو تصویری شخصی برای نقاشان و هنرجویان — دسته‌بندی بر اساس تکنیک، سبک و موضوع (نه نام هنرمند)، پوشه‌بندی نامحدود، علاقه‌مندی‌ها، گالری با چیدمان masonry (پرکردن فضای خالی مثل پینترست)، افزودن تصویر از فایل یا لینک، انتخاب چندتایی و اشتراک‌گذاری.

## تکنولوژی

- React 18 + Vite
- آیکون: lucide-react
- ذخیره‌سازی: `localStorage` (در اجرای مستقل) — همان کد بدون تغییر داخل Claude Artifacts هم با `window.storage` کار می‌کند (`src/storage.js`)

## اجرای محلی

```bash
npm install
npm run dev
```

سپس آدرس نمایش‌داده‌شده در ترمینال (معمولاً `http://localhost:5173`) را باز کن.

## Build برای production

```bash
npm run build
```

خروجی در پوشه‌ی `dist/` ساخته می‌شود.

## دیپلوی روی Cloudflare Pages

### روش ۱ — از طریق داشبورد Cloudflare (ساده‌ترین راه)

1. این ریپازیتوری را در گیت‌هاب push کن.
2. در Cloudflare Dashboard برو به **Workers & Pages → Create → Pages → Connect to Git** و همین ریپو را انتخاب کن.
3. تنظیمات build:
   - **Framework preset:** Vite
   - **Build command:** `npm run build`
   - **Build output directory:** `dist`
4. روی **Save and Deploy** بزن.

از این به بعد هر push به شاخه‌ی اصلی، به‌طور خودکار دوباره دیپلوی می‌شود.

### روش ۲ — از طریق CLI (wrangler)

```bash
npm install -g wrangler
npm run build
wrangler pages deploy dist --project-name painting-gallery
```

## نکته درباره‌ی داده‌ها

داده‌ها (پوشه‌ها، تنظیمات، تصاویر) در `localStorage` همان مرورگری که برنامه را باز می‌کنی ذخیره می‌شوند — یعنی محلی روی همان دستگاه/مرورگر هستند، نه در فضای ابری Cloudflare. اگر بعداً خواستی داده‌ها بین دستگاه‌ها sync شوند، باید یک بک‌اند/دیتابیس واقعی (مثلاً Cloudflare D1 یا KV) اضافه شود.

## نسخه فعلی (v0)

دسته‌بندی، پوشه‌سازی نامحدود، آپلود و فشرده‌سازی خودکار تصویر، افزودن با لینک، علاقه‌مندی‌ها، گالری masonry، انتخاب چندتایی و اشتراک‌گذاری، تنظیمات ظاهری (حالت شب + سه پوسته رنگی).

برای نسخه‌های بعد: تگ‌گذاری، جستجوی پیشرفته، یادداشت کنار تصویر، اتصال به دوره‌های آموزشی.
